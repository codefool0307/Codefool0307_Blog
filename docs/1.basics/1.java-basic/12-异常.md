## 12.0 异常综述

## 12.1 异常体系结构

异常本身就是一个类。产生异常就是创建了一个异常对象并抛出了一个异常对象

Java处理异常的方式就是中断处理

注：异常并不是语法错误

异常共分为了两个大类：
* Error
* Exception
