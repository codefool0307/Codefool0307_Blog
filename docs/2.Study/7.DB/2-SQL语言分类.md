#  SQL的语言分类
1. DQL（Data Query Language）：数据查询语言
		select 
2. DML(Data Manipulate Language):数据操作语言
		insert 、update、delete
3. DDL（Data Define Languge）：数据定义语言
		create、drop、alter
4. TCL（Transaction Control Language）：事务控制语言
		commit、rollback
